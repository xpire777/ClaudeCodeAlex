## Project Overview

## Git Workflow

- Repository: https://github.com/xpire777/ClaudeCodeAlex
- **Commit and push frequently** — after every meaningful piece of work, commit with a clean, direct message and push to GitHub immediately so progress is never lost
- Keep commits granular so reverting to any prior state is easy
- Do not batch large amounts of work into a single commit — break it up
